## DESCRIPTION 

This repo was created in target to get same experianse with #nestjs and to do Neobis task 

a simple TODO with CRUD api has 
Эндпоинты, которые должны быть:
- Список всех задач (GET)
- Вывод одной задачи (GET)
- Создание (POST)
- Редактирование (PUT)
- Удаление (DELETE)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
